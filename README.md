# Matrice LED - Last.fm Visualizer

Un simulateur Python qui affiche en temps réel les pochettes d'albums de ce que vous écoutez sur Spotify (via Last.fm). Ce projet est conçu pour piloter une matrice LED, mais il inclut un simulateur visuel pour tester l'affichage directement sur ton écran.

## Fonctionnalités
* Synchronisation en direct : Récupère automatiquement le titre et l'artiste en cours de lecture.
* Affichage dynamique : Télécharge et redimensionne les pochettes d'albums pour un affichage propre.
* Mode "Clear" : L'affichage se nettoie automatiquement quand la musique s'arrête.[cite: 3, 4]
* Simulation : Inclut un simulateur Pygame pour visualiser le résultat sans avoir besoin du matériel physique.[cite: 3, 4]
